# Retail Sales & Customer Analytics Dashboard

## Project Overview

This project analyzes 10K+ retail sales records using SQL, Python, and Power BI to identify customer behavior, sales trends, and profitability insights.

## Tech Stack

* Python
* SQL
* SQLite
* Pandas
* NumPy
* Power BI

## Features

* Data cleaning and preprocessing
* Customer analytics
* Product performance analysis
* Regional sales analysis
* KPI dashboards
* Forecasting insights

## Project Structure

```text
Retail-Sales-Analytics/
│
├── data/
├── python/
├── sql/
├── reports/
├── requirements.txt
└── run_pipeline.py
```

## KPIs

* Total Sales
* Total Profit
* Profit Margin
* Top Customers
* Regional Performance

## Business Insights

* Technology category generated highest revenue.
* West region showed strongest profitability.
* High discounts reduced profit margins.

## How to Run

```bash
pip install -r requirements.txt
python run_pipeline.py
```
